# agency-form-test-template

Starter replica of the workplace Playwright E2E repository structure.

## Included
- top-level repo layout
- E2E project under the e2e folder
- Playwright TypeScript starter setup
- Page Object Model starter files

## Next steps
1. Add your real environment values to the e2e .env file.
2. Install dependencies in the e2e folder.
3. Add your real page objects and tests.
